# 目录结构

目录结构如下：

```
├── apps                   # 子应用
│ ├── system               # 系统管理
│ └── example              # 示例应用
├── assets                 # 静态资源
├── cli                    # 脚手架
├── components             # 组件
├── main                   # 主应用
├── server                 # 服务端
├── package.json           # 依赖
├── pnpm-workspace.yaml    # 工作区配置
├── rsbuild.config.mjs     # 打包配置
└── README.md              # 说明
```
