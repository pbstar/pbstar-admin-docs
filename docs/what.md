# 什么是 PbstarAdmin?

Pbstar-admin 是一套基于微前端架构的后台管理系统解决方案，提供从开发到部署的全套工具链支持。

## 核心优势

- 🏗️ **微前端架构**：基于 Wujie 框架，支持多个子应用独立开发、测试和部署，主从应用无缝集成
- 🛠️ **CLI 脚手架**：提供交互式命令行工具，一键创建子应用，自动生成目录结构和配置文件
- 📚 **丰富组件库**：基于 Vue 3 + Element Plus，提供 p-title、p-form、p-table 等完整组件系统
- 💻 **现代化技术栈**：采用 Vue 3、pnpm、monorepo 工作区管理，提升开发效率和代码复用性
- 🎨 **灵活布局**：内置主题切换、全屏模式、响应式设计，完美适配各种设备和屏幕尺寸

## 微前端架构

采用 wujie + rsbuild + pnpm 技术组合实现微前端架构：

- **wujie**：无界微前端框架，实现应用隔离和通信
- **rsbuild**：基于 Rspack 的高性能构建工具
- **pnpm**：高效的包管理工具，支持模块独立依赖

## 主流技术栈

- Vue 3：采用 Composition API 和 `<script setup>` 语法
- Pinia：轻量级状态管理库
- Element Plus：基于 Vue 3 的组件库
